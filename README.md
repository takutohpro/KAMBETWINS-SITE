# KAMBETWINS GALLERY

**KAMBE TWINS**（神戸あい・神戸あお）の公式ポートフォリオ＆ギャラリーサイト。

## 技術スタック

- HTML5
- CSS3（グラスモーフィズム、レスポンシブ対応）
- JavaScript（Vanilla JS）

## 構成

| セクション | 内容 |
|------------|------|
| HOME | ヒーローセクション |
| ABOUT | プロジェクトの概要 |
| PROFILE | プロフィール紹介（神戸あい・神戸あお） |
| GALLERY | イラストギャラリー（ライトボックス表示） |
| LINK | X / Patreon のリンク |

※ サイト内はヘッダーあるいはメニューのボタン（EN / JP）で言語切り替えが可能です。

## ローカルでの確認

`index.html` をブラウザで直接開くか、ローカルサーバーで起動してください。

```bash
# Python の場合
python -m http.server 8000

# Node.js の場合
npx serve .
```
